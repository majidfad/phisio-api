# Push to GitHub

Repos are committed locally. Run these commands after `gh auth login`:

```powershell
gh repo create majidfad/phisio-api --public --confirm
cd c:\Users\Mahboubeh\source\repos\Phisio.Backend
git push -u origin main
```

Or create an empty repo at https://github.com/new named `phisio-api`, then `git push -u origin main`.
